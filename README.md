\# DNKConnect



\## Description

DNKConnect is a full-stack e-commerce web application developed using Java, JSP, and Servlets.

The project follows MVC architecture and uses Maven for dependency management.



\## Tech Stack

\- Java

\- JSP, Servlets

\- HTML, CSS

\- MySQL / PostgreSQL

\- Maven

\- MVC Architecture



\## Features

\- User authentication and session management

\- Product listing and cart functionality

\- Order placement and management

\- Admin-side product and category management

\- Backend request handling using Servlets



\## Architecture

\- Model: Entity and DAO classes

\- View: JSP pages

\- Controller: Servlets handling requests and responses



\## How to Run

1\. Clone the repository

2\. Import as a Maven project in Eclipse

3\. Configure the database

4\. Deploy on Apache Tomcat
